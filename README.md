# Cybersecurity Learning Repository

Este repositorio recoge mi proceso de aprendizaje en ciberseguridad, con prácticas, notas y pequeños proyectos realizados mientras estudio fundamentos de seguridad informática.

Actualmente estoy formándome en ciberseguridad tras completar ASIR, y este repositorio sirve como portfolio práctico donde documento lo que voy aprendiendo.

## Contenido del repositorio

### TryHackMe
Prácticas y laboratorios realizados en la plataforma TryHackMe, organizados por rutas y niveles.
- Fundamentos de redes
- Linux básico
- Introducción a la ciberseguridad
- Conceptos ofensivos y defensivos

Ruta actual:
- `tryhackme/pre-security`

### Scripts
Pequeños scripts desarrollados con fines educativos para automatizar tareas básicas relacionadas con redes y seguridad.
- Escaneos básicos
- Pruebas y utilidades simples

Ruta:
- `scripts/`

## Objetivo
- Consolidar conocimientos prácticos en ciberseguridad
- Acompañar la preparación de certificaciones como CompTIA Security+
- Construir un portfolio técnico visible y bien documentado

## Nota
Todo el contenido es educativo y se ha realizado en entornos controlados (labs y máquinas de práctica).
