# Workplace Security Policies / Políticas de Seguridad en el Puesto de Trabajo

## Español

Esta carpeta contiene las políticas de seguridad diseñadas para los distintos puestos de trabajo de la organización.  
Incluye análisis de riesgos, medidas de protección y buenas prácticas para directivos, personal administrativo y personal de la tienda online.

---

## English

This folder contains the security policies designed for different workplace positions.  
It includes risk analysis, protective measures, and best practices for executives, administrative staff, and online store employees.
