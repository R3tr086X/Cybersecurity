# Network Subnetting Simulation / Simulación de Subnetting de Red

---

## Español

### Descripción

Documentación del diseño, configuración y validación de una red en Cisco Packet Tracer. Se configuraron múltiples dispositivos con direcciones IP, máscaras de red y puertas de enlace. Además, se aplicó subnetting VLSM para dividir la red en subredes optimizadas para distintos departamentos.

### Red Principal y Subredes

- Red principal: Segmentada en varias subredes para Finanzas, Administración e Informática.  
- Subredes configuradas con la máscara mínima necesaria según la cantidad de dispositivos.  
- Se incluyó una subred adicional para interconectar routers.  
- IPs consecutivas asignadas siguiendo criterios de optimización y eficiencia.

### Configuración de Dispositivos

- Router principal: interfaces activadas y configuradas con IPs de subred correspondientes.  
- Switches conectando PCs de cada subred.  
- PCs configurados con IP estática, máscara de subred y puerta de enlace correcta.  
- Conectividad verificada con pings entre todos los equipos.

## English

### Description

Documentation of the design, configuration, and verification of a network in Cisco Packet Tracer. Multiple devices were configured with IP addresses, subnets, and gateways. VLSM subnetting was applied to divide the network into optimized subnets for different departments.

### Main Network and Subnets

- Main network segmented into Finance, Administration, and IT subnets.  
- Subnets sized according to the minimum number of hosts required.  
- Additional subnet created for router interconnection.  
- Consecutive IP ranges assigned for efficiency.

### Device Configuration

- Main router: interfaces activated and assigned subnet IPs.  
- Switches connecting PCs for each subnet.  
- PCs configured with static IP, subnet mask, and correct gateway.  
- Network connectivity verified via ping across all devices.
